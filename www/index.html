Insitu-x

Free Insitux pseudo-FaaS platform for Discord bots and web apps

Visit the Discord server to discuss Insitux, and have your own Insitux code deployed as a Discord bot handler and/or web-app. Use the Insitux Discord Bot invite if you'd like to use apps in your own server. It can also run Insitux code directly in codeblocks.

How it works

Apps are single Insitux code files. The Discord bot responds to either Insitux codeblocks or valid application prefixes. If a codeblock, it will immediately execute the code. If a prefix, it downloads the app's latest source code, and invokes its handler function, responding with the function's return. If you visit an app's website endpoint it will make a GET request to the app in the same fashion.

Interactions

Apps will either respond to Discord messages or visiting their endpoint on this website.

To invoke in Discord, use their prefix in a server with the Insitux Discord Bot present, e.g. "!app-name [your message here]"

To invoke their GET/POST handler, visit https://insitux.repl.co/app-name in your browser.
