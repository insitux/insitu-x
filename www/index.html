Insitu-x

Free Insitux FaaS platform for Discord bots and web apps

Visit the Discord server to discuss Insitux, and have your own Insitux code deployed as a Discord bot handler and/or web-app.

Use this Insitux Discord Bot invite if you'd like to use apps in your own server. It can also run Insitux code directly in codeblocks.

How it works

The Discord bot picks up either Insitux codeblocks or valid application prefixes. If a codeblock it will immediately execute the code. If a prefix it calls the POST API sat on the same URL you are reading now, downloads the app's latest source code, and invokes its handler function. The bot then responds to the original message with whatever the handler function returns.

Currently deployed apps

Apps will either respond to Discord messages or visiting their endpoint on this website.

To invoke in Discord, use their prefix in a server with the Insitux Discord Bot present, e.g. "!app-name [message to send to the app]"

To invoke their GET/POST handler, visit https://insitux.repl.co/app-name in your browser.
